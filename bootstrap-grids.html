Bootstrap Grid Sandbox

Bootstrap uses a 12 column responsive grid. Bootstrap achieves this responsiveness with the use of classes adhering to an easy to comprehend naming convention that cotrol the width of the element on which the class is applied. The responsive nature cause the column to revert back to 100% width for browser window sizes smaller than the pixel ratios listed below (using the smallest of all applied classes). These classes include:

SIZECLASSWINDOW SIZE
Extra Small.col-xs-%0 and above
Small.col-sm-%768px and above
Medium.col-md-%992px and above
Large.col-lg-%1200px and above

Chart Example 1

This is an example of a chart styled with Bootstrap's grid system. The conatiner is 8 parts wide (grids are 12 parts). It has an offset of 2. Offsets add spacing to the left of a container. In order to center a div using the grid system, you take 12 (Grid width) - 8 (container width) to get 4 remaining. Divide this by two (1/2 on the left and 1/2 on the right).
